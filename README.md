# TP Fullstack - Mist project
## Explications du projet
__Mist__ est une plateforme de __distribution__ de jeux vidéo, inspirée de Steam. <br>
Cette plateforme permet aux utilisateurs de:
- parcourir un catalogue public de jeux vidéo
- acquérir un ou plusieurs jeux au sein de ce catalogue
- gérer leur bibliothèque de jeux acquis
